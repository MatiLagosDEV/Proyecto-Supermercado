# Proyecto-Supermercado

Aplicación orientada a **Android y iOS** para comparar precios de supermercados.

## Tecnologías utilizadas

| Capa                  | Tecnología                                      |
|-----------------------|------------------------------------------------|
| Frontend móvil        | React Native (iOS + Android)                  |
| Backend / API         | Python + FastAPI                               |
| Scraping              | Python (Scrapy, BeautifulSoup, Playwright)    |
| Base de datos         | PostgreSQL o MongoDB                           |
| Infraestructura       | AWS / Heroku / DigitalOcean + S3 + Cron/Celery|

## Descripción

Este proyecto permite a los usuarios **comparar precios de distintos supermercados** mediante un backend que realiza scraping de los sitios web y entrega los datos a la app móvil a través de una API.  

El stack elegido garantiza:

- Desarrollo rápido y multiplataforma.
- Escalabilidad y fácil mantenimiento.
- Recolección de datos automatizada y eficiente.
